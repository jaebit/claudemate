# /cw:check is now /cw:review --compliance

This command was consolidated into `/cw:review` in CW v3.0.

**Migration:**
- `/cw:check` → `/cw:review --compliance`
- `/cw:check --rules` → `/cw:review --compliance --rules`
- `/cw:check --docs` → `/cw:review --compliance --docs`
- `/cw:check --all` → `/cw:review --compliance --all`

This stub will be removed in v3.1.
