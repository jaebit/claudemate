# /cw:research is now /cw:explore --research

This command was consolidated into `/cw:explore` in CW v3.0.

**Migration:**
- `/cw:research "topic"` → `/cw:explore --research "topic"`
- `/cw:research --internal` → `/cw:explore --research --internal`
- `/cw:research --external` → `/cw:explore --research --external`
- `/cw:research --depth deep` → `/cw:explore --research --depth deep`

This stub will be removed in v3.1.
