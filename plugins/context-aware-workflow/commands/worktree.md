# /cw:worktree is now /cw:manage worktree

This command was consolidated into `/cw:manage` in CW v3.0.

**Migration:**
- `/cw:worktree create phase 2` → `/cw:manage worktree create phase 2`
- `/cw:worktree list` → `/cw:manage worktree list`
- `/cw:worktree clean` → `/cw:manage worktree clean`

This stub will be removed in v3.1.
