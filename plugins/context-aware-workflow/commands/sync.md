# /cw:sync is now /cw:manage sync

This command was consolidated into `/cw:manage` in CW v3.0.

- `/cw:sync` → `/cw:manage sync`
- `/cw:sync --to-serena` → `/cw:manage sync --to-serena`
- `/cw:sync --from-serena` → `/cw:manage sync --from-serena`
- `/cw:sync --status` → `/cw:manage sync --status`

This stub will be removed in v3.1.
