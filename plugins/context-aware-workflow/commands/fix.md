# /cw:fix is now /cw:review --fix

This command was consolidated into `/cw:review` in CW v3.0.

- `/cw:fix` → `/cw:review --fix`
- `/cw:fix --deep` → `/cw:review --fix --deep`
- `/cw:fix --interactive` → `/cw:review --fix --interactive`
- `/cw:fix --dry-run` → `/cw:review --fix --dry-run`

This stub will be removed in v3.1.
