# /cw:evolve is now /cw:manage evolve

This command was consolidated into `/cw:manage` in CW v3.0.

**Migration:**
- `/cw:evolve` → `/cw:manage evolve`
- `/cw:evolve --preview` → `/cw:manage evolve --preview`
- `/cw:evolve --create command` → `/cw:manage evolve --create command`
- `/cw:evolve --id <id>` → `/cw:manage evolve --id <id>`

This stub will be removed in v3.1.
